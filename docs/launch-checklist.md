## Section 2: Page Set (7 Pages)

All 7 pages must be present at the production URL.

- [ ] **Home** — Primary landing page with candidate intro, platform highlights, and conversion CTAs
- [ ] **Issues and Answers** — Detailed policy positions on priority issues
- [ ] **Donate** — Embedded donation form from the candidate's fundraising platform
- [ ] **Contact** — Campaign contact details, social links, and scheduling element
- [ ] **About** — Candidate biography, credentials, and background
- [ ] **Privacy Policy** — Privacy disclosures populated from intake data
- [ ] **Cookie and SMS Compliance Policy** — Cookie/tracking/TCPA disclosures populated from intake data

---

## Section 3: Site-Wide Components

All site-wide components must be operational across all 7 pages.

- [ ] **3a. Responsive layout** — Renders correctly on desktop (1440px), tablet (768px), and mobile (375px)
- [ ] **3b. Navigation** — Links to all 7 pages (Home, About, Issues, Donate, Contact, Privacy, Cookie). Works on desktop and mobile.
- [ ] **3c. Footer** — Present on every page. Contains:
  - [ ] Paid-for-by disclaimer with committee name
  - [ ] Committee mailing address
  - [ ] Social media links (if provided during intake)
  - [ ] Link to Privacy Policy page
  - [ ] Link to Cookie and SMS Compliance Policy page
- [ ] **3d. Donate CTA** — Donate button accessible from navigation on every page
- [ ] **3e. Voter survey embed** — One survey present per intake selection (page, homepage section, or both)
- [ ] **3f. Visual theme** — Candidate logo, colors, and typography applied throughout
- [ ] **3g. SEO metadata** — Page title and meta description set on every page
- [ ] **3h. White-labeled admin** — Login page, admin bar, and dashboard show client branding

## Section 4: Compliance Content

Both compliance pages must be present and populated with intake data.

### 4.1 Privacy Policy

- [ ] Categories of personal information collected
- [ ] Purposes for collection, use, and sharing
- [ ] Cookie and tracking technology disclosures
- [ ] SMS/TCPA disclosures (if SMS program indicated)
- [ ] Data retention practices
- [ ] State privacy rights (CA, CO, CT, FL, OR, TX, UT, VA)
- [ ] Contact information for privacy inquiries
- [ ] Committee name correctly populated
- [ ] Mailing address correctly populated
- [ ] Email and phone correctly populated
- [ ] Website URL correctly populated
- [ ] Effective date set

### 4.2 Cookie and SMS Compliance Policy

- [ ] Cookie categories described (necessary, analytics, functionality, advertising)
- [ ] Purpose of each category explained
- [ ] Consent management and opt-out mechanisms
- [ ] 10DLC and TCPA disclosures present
- [ ] STOP and HELP keyword behavior described
- [ ] Committee name correctly populated
- [ ] Contact information correctly populated
- [ ] Effective date set

### 4.3 Site-Wide Compliance Notices

- [ ] Footer paid-for-by disclaimer matches `legal_compliance.paid_for_by` from intake
- [ ] Committee name and address in footer
- [ ] Privacy Policy link in footer works
- [ ] Cookie/SMS Policy link in footer works
